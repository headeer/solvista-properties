A custom WordPress theme for real estate properties with interactive map and advanced filtering capabilities.

== Description ==

Solvista Property is a specialized WordPress theme designed for real estate websites. It features an interactive map interface with advanced property filtering capabilities.

= Key Features =

* Interactive property map using Leaflet.js
* Advanced property filtering system
* Custom property post type
* Responsive design
* Quick selection filters
* Detailed property information display

== Installation ==

1. Upload the theme files to the `/wp-content/themes/solvista-property` directory, or install the theme through the WordPress themes screen directly.
2. Activate the theme through the 'Themes' screen in WordPress.
3. Create a new page and select "Main Page with Map" as the template.
4. Set this page as your homepage in Settings > Reading.

== Frequently Asked Questions ==

= Does this theme support any plugins? =

Solvista Property is designed to work with the WordPress core functionality. It includes its own custom post type for properties.
